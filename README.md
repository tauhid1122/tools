# Tools Collection

A collection of useful web-based tools built with HTML, CSS, and JavaScript.

## Features

- Character Counter
- Color Picker
- Calculator

## Structure

```
tools-collection/
│
├── assets/
│   ├── images/
│   ├── icons/
│   └── fonts/
│
├── css/
│   ├── style.css
│   └── components/
│       ├── counter.css
│       ├── colorpicker.css
│       └── calculator.css
│
├── js/
│   ├── main.js
│   └── tools/
│       ├── counter.js
│       ├── colorpicker.js
│       └── calculator.js
│
├── index.html
└── README.md
```

## Setup

1. Clone the repository
2. Open index.html in your browser
3. Start using the tools!
